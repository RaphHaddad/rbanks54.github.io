Google has released the Code Prettifier - a little bit of javascript and css to turn unformatted code into something nice:

For example

void Main()
{
  Console.WriteLine("Hello Blog!!");
}


Can become

void Main()
{
  Console.WriteLine("Hello Blog!!");
}

With just the following tags around the code <pre class="prettyprint">...</pre> or <code class="prettyprint">...</code>

Thanks Google!
